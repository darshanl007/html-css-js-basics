Loops in Javascript

for loop : Used when you know how many times you want to run the loop.

while loop : Runs as long as the condition is true.

do-while loop : Runs at least once, even the condition is false.

for-of loop : Used to iterate over values of arrays,strings, etc.

for-in loop : Used to iterate over object keys and aslo access their corresponding values.

break : Stops the loop immediately

continue : Skips the current iteration
